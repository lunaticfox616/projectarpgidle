# 패치 노트

## v0.9.3 — 2026-06-11
- 소환수 피해 대폭 상향:
  - 공격 소환수 기본 피해를 약 1.5배 상향했습니다.
  - 소환수 젬 레벨에 따른 피해 성장 배율을 증가시켰습니다(고레벨일수록 더 큰 폭).
  - 소환수에 적용되던 '최종 피해 배율 축소'를 제거해 플레이어 최종 피해 배율을 100% 적용받습니다.
  - 소환수에 적용되던 저항 관통 제한을 완화했습니다(플레이어 관통 35%→55% 반영, 상한 25→40 / 물리 무시 20→30).
  - 모든 공격 소환수에 기본 치명타 확률 5% 하한을 보장합니다.
- (소환수 관련 스탯: 소환수 피해/추가 피해/효율/치명타/치명타 피해/저항 관통/생명력은 기존에도 정상 적용되고 있었습니다.)

## v0.9.2 — 2026-06-11
- 소울바인더 '상호 보완'(sb7) 키스톤이 '피해 증가 수치'가 아닌 실제 '공격력(타격당 기본 피해)'을 기준으로 공유하도록 변경했습니다. 플레이어 공격력의 50%가 소환수 타격에, 대표 소환수 공격력의 50%가 플레이어 타격에 더해집니다.
- 각 측의 공격력은 상대의 보너스를 제외한 자기 스탯으로만 계산하여 무한 피드백 루프 없이 동작합니다(기존 의도 유지).
- 캐릭터 탭의 공격력/DPS 커스텀 툴팁과 소환 DPS 툴팁에 상호 보완으로 더해지는 공격력 수치를 표기합니다.

## v0.9.1 — 2026-06-11
- 창공 각인 '각성: 심층 초월'(젬 레벨 +3)이 소환수 공격 젬에도 정상 적용됩니다.
- 스킬 젬 강화 탭은 한 번 개방하면 다음 루프에도 계속 열린 상태로 유지됩니다.
- 각인 해제 기능을 다시 사용할 수 있습니다. 적용 중인 각인 버튼을 다시 누르면 창공의 힘을 소모해 해제하며, 젬 각인사 Lv.7부터는 무료로 해제됩니다.
- 소환수가 여러 마리일 때도 DPS 계산과 실제 공격이 마릿수만큼 정확히 반영되는지 검증을 추가했습니다.

## v0.9.0 — 2026-06-10
- 전투 화면에서 플레이어 또는 몬스터가 회피하면 "회피!" 텍스트가 표시됩니다.
- 빗겨낸 피해는 대미지 숫자 색상(연녹색)과 전투 로그로 구분해 표시합니다.
- 설정 탭과 시작 화면에 패치 노트를 추가했습니다.
