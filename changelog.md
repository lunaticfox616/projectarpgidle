# 패치 노트

## v0.9.1 — 2026-06-11
- 소환수 피해 상향: 기본 피해·레벨 성장 배율 증가, 최종 피해 배율 축소 제거, 저항 관통 제한 완화, 기본 치명타 5% 보장.
- 소울바인더 '상호 보완'이 피해 증가가 적용된 실제 공격력 기준으로 공유되도록 변경(무한 루프 없음). 관련 수치를 공격력/DPS 툴팁에 표기.
- 창공 각인 '심층 초월'(젬 레벨 +3)이 소환수 공격 젬에도 적용됩니다.
- 스킬 젬 강화 탭이 한 번 개방되면 다음 루프에도 유지됩니다.
- 각인 해제 재활성화: 각인 버튼을 다시 누르면 창공의 힘으로 해제(젬 각인사 Lv.7부터 무료).
- 패치 노트를 '패치 노트 보기' 버튼의 스크롤 팝업으로 표시하도록 변경.

## v0.9.0 — 2026-06-10
- 전투 화면에서 플레이어 또는 몬스터가 회피하면 "회피!" 텍스트가 표시됩니다.
- 빗겨낸 피해는 대미지 숫자 색상(연녹색)과 전투 로그로 구분해 표시합니다.
- 설정 탭과 시작 화면에 패치 노트를 추가했습니다.
