# 패치 노트

## v0.9.1 — 2026-06-11
- 창공 각인 '각성: 심층 초월'(젬 레벨 +3)이 소환수 공격 젬에도 정상 적용됩니다.
- 스킬 젬 강화 탭은 한 번 개방하면 다음 루프에도 계속 열린 상태로 유지됩니다.
- 각인 해제 기능을 다시 사용할 수 있습니다. 적용 중인 각인 버튼을 다시 누르면 창공의 힘을 소모해 해제하며, 젬 각인사 Lv.7부터는 무료로 해제됩니다.
- 소환수가 여러 마리일 때도 DPS 계산과 실제 공격이 마릿수만큼 정확히 반영되는지 검증을 추가했습니다.

## v0.9.0 — 2026-06-10
- 전투 화면에서 플레이어 또는 몬스터가 회피하면 "회피!" 텍스트가 표시됩니다.
- 빗겨낸 피해는 대미지 숫자 색상(연녹색)과 전투 로그로 구분해 표시합니다.
- 설정 탭과 시작 화면에 패치 노트를 추가했습니다.
